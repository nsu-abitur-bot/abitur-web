## Подготовка

1. Скопируйте файл с переменными окружения:

```bash
cp .env.example .env
```

2. Заполните `.env` (адрес бэкенда и другие настройки).

## Docker

```bash
docker-compose up -d --build
```

После запуска, приложение доступно по адресу `http://localhost:3000`.

## Запуск в dev режиме

1. Установить зависимости:

```bash
pnpm i
```

2. Запустить приложение (по умолчанию `http://localhost:7868` или другой порт из `.env`):

```bash
pnpm dev
```
